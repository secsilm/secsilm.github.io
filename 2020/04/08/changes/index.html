现在我有三个博客：

  1. Alan Lee，独立博客
  2. The North Cloud Gate，独立博客
  3. CSDN
  4. 知乎

之前我的技术类文章都是在 1 和 3 上同步发表，造成了严重的信息冗余，博客划分不够细。因此接下来一段时间我会按照如下的划分，来调整博客上的文章：

  1. Alan Lee：主要用来分享 NLP/DL/ML 的相关的技术文章。所以其他类别的文章，应该全部会转移到 3 上。
  2. The North Cloud Gate：生活类博客，旅行、俩喵等等。
  3. CSDN：除 1 所涉及类别（NLP/DL/ML）外的所有技术类文章，主要包括 Python 相关技术（编程、可视化、坑点等）、Linux 相关和其他有的没的的折腾。那么 NLP/DL/ML 类的文章，除少数访问量大的文章外，应该都会全部转移到 1 上。
  4. 知乎：1 的部分会发表到这里。
