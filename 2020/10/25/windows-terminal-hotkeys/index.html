Windows Terminal 快捷键

2020 年 10 月 25 日

目录

  1. Reference

这里记录一下常用的 Windows terminal 快捷键。

序号 快捷键 说明
1 Alt + Shift + = 横向 split
2 Alt + Shift + - 纵向 split
3 Alt + 方向键 在各个 panes 之间切换
4 Alt + Shift + 方向键 调整 pane 的大小
5 Ctrl + Shift + W 关闭当前 pane，注意如果是最后一个的话会直接关闭整个 terminal 窗口
